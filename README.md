# Jefferson Cell - Meu Caixa PRO

Aplicativo de gestão de assistência técnica para celulares com suporte offline.

## 🚀 Acesso

### Versão Web (GitHub Pages)
Acesse: https://jefferson6888.github.io/AppMeuCaixa3

### Versão Mobile (APK)
- Baixe o APK para Android
- Funciona 100% offline
- Dados sincronizados localmente

## ✨ Funcionalidades

- ✅ Gerenciamento de consertos
- ✅ Controle de vendas
- ✅ Histórico de garantias
- ✅ Dashboard com estatísticas
- ✅ Funcionamento offline
- ✅ Backup automático de dados

## 📱 Compatibilidade

- **Web**: Qualquer navegador moderno (Chrome, Firefox, Safari, Edge)
- **Mobile**: Android 5.0+ (via APK)

## 💾 Dados

Todos os dados são armazenados localmente no seu dispositivo/navegador. Nenhuma informação é enviada para servidores externos.
